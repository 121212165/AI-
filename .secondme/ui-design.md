# AI 戒酒互助会 - UI 设计方案

## 设计原则
- **亮色主题**：温暖、支持性的色调
- **简约现代**：清晰的信息层级
- **中文界面**：所有文字使用中文
- **稳定优先**：简单过渡动画

## 色彩方案
```
主色：绿色 #10B981 （希望、成长）
辅助色：橙色 #F59E0B （温暖、鼓励）
背景色：浅灰 #F9FAFB
文字色：深灰 #111827
```

## 组件列表

### 1. 布局组件
- `Layout.tsx` - 主布局（导航栏、页面容器）
- `Header.tsx` - 顶部导航
- `Footer.tsx` - 底部信息

### 2. 用户组件
- `UserCard.tsx` - 用户卡片（头像、昵称、已戒天数）
- `LoginButton.tsx` - 登录按钮
- `UserProfile.tsx` - 用户资料页

### 3. 互动组件
- `CheckInButton.tsx` - 打卡按钮
- `CrisisButton.tsx` - 危机求助按钮（大、醒目）
- `EncouragementCard.tsx` - 鼓励消息卡片
- `MessageList.tsx` - 消息列表

### 4. 展示组件
- `StatsCard.tsx` - 统计卡片
- `Leaderboard.tsx` - 排行榜
- `AchievementBadge.tsx` - 成就徽章
- `ProgressRing.tsx` - 进度环

### 5. 表单组件
- `CheckInForm.tsx` - 打卡表单
- `NicknameForm.tsx` - 昵称设置表单
